# MVG Departures Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Real-time departure information for Munich public transport (U-Bahn, S-Bahn, Tram, Bus)
- Grouped departures by transport type for easy navigation
- Searchable station dropdown with all Munich stations
- Quick access to home and work stations via local storage
- Visual delay indicators and cancellation alerts
- Platform information and occupancy levels
- Auto-refresh every 30 seconds for current data
- Actions to set home and work stations via searchable interface
